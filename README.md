# 🤝 BlueHand (Helping Hand)

BlueHand is a mobile application designed to connect people who need help with nearby individuals willing to assist. The platform focuses on simplifying everyday tasks by building a local support network.

---

## 🚀 Features

- 📍 Location-based service (nearby helpers)
- 🧑‍🤝‍🧑 Connect people for household and daily tasks
- 📦 Request and offer services easily
- 🔐 User authentication (Login/Signup)
- 📡 Real-time request handling
- 🗺️ Map integration (OSM / OpenStreetMap)

---

## 🛠️ Tech Stack

- **Frontend:** Android (Java/Kotlin)
- **Backend:** Firebase / API-based (if used)
- **Database:** Firebase Realtime DB / Firestore
- **Maps:** OpenStreetMap (OSM)
- **IDE:** Android Studio
